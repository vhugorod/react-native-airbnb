# react-native-airbnb-clone
 Airbnb clone app using React Native & Redux
 
# How to use
 <li>Clone</li>
 Run npm install in the terminal
 Run on IOS or Android projects in emulator

# Dependencies
 react
 react-native
 react-native-vector-icons
 react-navigation
 react-redux
 redux
 redux-logger
 redux-thunk
